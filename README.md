## Запуск проекта

Клонируем репозиторий
~~~bash  
  git clone https://link-to-project
~~~

Переходим в репозиторий
~~~bash  
  cd my-project
~~~

Создаем виртуальное окужение & устанавливаем зависимости
~~~bash  
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
~~~

Запустить
~~~bash  
python main.py
~~~
